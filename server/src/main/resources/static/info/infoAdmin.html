Автоматизированная система составления и разгадывания классического кроссворда по выбранной теме.

Оглавление

Функции администратора.

Технические требования к системе.

Руководство для пользователя:

- Навигационная панель.

- Главная страница.

- Страница работы с кроссвордами.

- Страница настройки параметров.

- Страница ручного составления кроссворда.

- Страница автоматической генерации кроссворда.

- Страница работы со словарями понятий.

- Страница загрузки словаря понятий.

- Страница создания словаря понятий.

- Страница изменения словаря понятий.

Функции администратора:

  • настройка параметров кроссворда при создании:
    1. задание размеров сетки;
    2. подключение словаря понятий;
  • составление/редактирование кроссворда:
    1. добавление слова;
    2. удаление слова;
    3. фильтрация словаря понятий по маске;
    4. сортировка словаря понятий;
  • сохранение кроссворда в файл заданной структуры;
  • работа со словарями понятий:
    1. добавление понятия;
    2. удаление понятия;
    3. выбор критерия сортировки;
    4. поиск по заданной маске;
    5. загрузка словаря из файла;
    6. сохранение словаря в файл;
    7. создание нового словаря понятий;

К оглавлению

Технические требования к системе:

  1. режим работы диалоговый;
  2. время автоматической генерации кроссворда – не более 10 с;
  3. система должна удовлетворять санитарным правилам и нормам СанПин 2.2.2./2.4.2198-07;
  4. условия работы средств вычислительной техники (содержание вредных веществ, пыли и подвижность воздуха) должны соответствовать ГОСТ 12.1.005, 12.01.007;
  5. температура окружающего воздуха – 15-35°С;
  6. влажность воздуха – 45-75%.

К оглавлению

Руководство для пользователя:

Навигационная панель.

На навигационной панели есть кнопки для перехода на страницу работы с кроссвордами, страницу работы со словарями понятий, страницу справки (сведения о разработчиках и о системе) и выход со страницы администратора.

К оглавлению

Главная страница.

На главной странице есть кнопки перехода на страницу работы с кроссвордами и на страницу работы со словарями понятий.

К оглавлению

Страница работы с кроссвордами.

Чтобы создать новый кроссворд, нужно нажать на кнопку «Создать новый кроссворд».

Ниже кнопки создания представлены уже созданные кроссворды. Чтобы изменить уже созданный кроссворд, нужно щелкнуть ЛКМ по нему.

Также можно загрузить кроссворд из файла и изменить его, для этого нужно нажать на кнопку «Из файла» и выбрать нужный файл. Загрузить можно только файл с расширением .kros.

К оглавлению

Страница настройки параметров.

После нажатия кнопки «Создать новый кроссворд» происходит переход на данную страницу.

Чтобы выбрать размеры (ширину и высоту) сетки, нужно воспользоваться стрелками вверх и вниз для увеличения и уменьшения значения соответственно.

Чтобы выбрать словарь понятий, слова из которого будут использованы при создании кроссворда, нужно нажать на стрелку рядом с названием словаря и в выпавшем списке выбрать нужный словарь, щелкнув ЛКМ по нему.

Чтобы выбрать вид составления кроссворда, нужно нажать на стрелку рядом с видом и в выпавшем списке выбрать нужный вид, щелкнув ЛКМ по нему. При автоматическом виде составления система сама создаст кроссворд из слов выбранного словаря понятий. При ручном виде составления, необходимо самостоятельно составить кроссворд вручную.

К оглавлению

Страница ручного составления кроссворда.

На данной странице присутствует сетка для составления кроссворда, справа указано название выбранного словаря, панель для отображения понятий из словаря, кнопки сортировки слов из словаря по алфавиту или по длине слов. Снизу расположено поле для ввода названия кроссворда. Также снизу расположены кнопки сохранения, скачивания и завершения.

Для того чтобы добавить слово на сетку, нужно выбрать область на сетке, после чего в панели понятий должны отобразиться подходящие слова для выбранной области, далее нужно выбрать слово из списка, щелкнув ЛКМ по нему.

Для того чтобы удалить слово с сетки кроссворда, нужно щелкнуть по нему ЛКМ на сетке, после чего нажать на кнопку Delete.

Чтобы сохранить кроссворд на данном этапе составления, нужно нажать кнопку «Сохранить». При этом кроссворд не будет сохранен на устройство. Но после этого кроссворд в текущем состоянии будет отображаться в списке созданных кроссвордов на странице работы с кроссвордами и в каталоге пользователя для разгадывания.

Чтобы скачать кроссворд на устройство, необходимо нажать кнопку «Скачать». Далее откроется окно скачивания файла, там нужно выбрать путь сохранение и нажать кнопку «Скачать».

К оглавлению

Страница автоматической генерации кроссворда.

На данной странице присутствует сетка со сгенерированным кроссвордом, справа указано название выбранного словаря, панель для отображения понятий из словаря. Снизу расположено поле для ввода названия кроссворда. Также снизу расположены кнопки сохранения, скачивания и завершения.

Чтобы сохранить кроссворд, нужно нажать кнопку «Сохранить». При этом кроссворд не будет сохранен на устройство. Но после этого кроссворд будет отображаться в списке созданных кроссвордов на странице работы с кроссвордами и в каталоге пользователя для разгадывания.

Чтобы скачать кроссворд на устройство, необходимо нажать кнопку «Скачать». Далее откроется окно скачивания файла, там нужно выбрать путь сохранение и нажать кнопку «Скачать».

К оглавлению

Страница работы со словарями понятий.

Это страница приветствия. Кроме того на этой странице и на всех страницах работы со словарями понятий присутствуют кнопки для перехода на страницы создания, загрузки и изменения словарей понятий.

К оглавлению

Страница загрузки словаря понятий.

Чтобы загрузить словарь из файла, нужно нажать кнопку выбора словаря и в открывшемся окне выбрать нужный файл. Загрузить можно только файл с расширением .dict. Далее нужно ввести название словаря, которое будет отображаться в системе, и нажать на кнопку «Загрузить».

К оглавлению

Страница создания словаря понятий.

Чтобы создать словарь понятий, нужно сначала ввести названия в соответствующее поле. Затем добавить понятие с определением, нажав кнопку «Добавить». В соответствующих полях нужно ввести слово и определение и нажать кнопку подтверждения. Слова не должны повторяться. Вводить можно только буквы. После добавления в таблице появится понятие и его определение.

Чтобы удалить понятие и определение, нужно нажать на кнопку-крестик в строке данного понятия.

После завершения работы над словарем, необходимо нажать кнопку «Сохранить», после чего словарь будет создан и будет отображаться в списках словарей.

К оглавлению

Страница изменения словаря понятий.

На данной странице можно изменить уже созданные словари. Чтобы выбрать словарь для изменения, нужно нажать на стрелку рядом с названием словаря и в выпавшем списке выбрать нужный словарь, щелкнув ЛКМ по нему.

Чтобы добавить понятие с определением, нужно нажать кнопку «Добавить». В открывшемся окне нужно ввести слово и определение и нажать кнопку подтверждения. Слова не должны повторяться. Вводить можно только буквы. После добавления в таблице появится понятие и его определение.

Чтобы удалить понятие и определение, нужно нажать на кнопку-крестик в строке данного понятия.

После завершения работы над словарем, необходимо нажать кнопку «Сохранить», после чего словарь будет создан и будет отображаться в списках словарей.

Чтобы скачать словарь на устройство, необходимо нажать кнопку «Скачать». Далее откроется окно скачивания файла, там нужно выбрать путь сохранение и нажать кнопку «Скачать».
