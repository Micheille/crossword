Автоматизированная система составления и разгадывания классического кроссворда по выбранной теме.

Оглавление

Функции администратора.

Технические требования к системе.

Руководство для пользователя:

- Навигационная панель.

- Главная страница (каталог)

- Страница разгадывания.

- Страница результатов.

- Вход.

Функции игрока:

  • разгадывание кроссворда: вписывание и удаление букв;
  • сохранение кроссворда в файл;
  • загрузка кроссворда из файла.

К оглавлению

Технические требования к системе:

  1. режим работы диалоговый;
  2. время автоматической генерации кроссворда – не более 10 с;
  3. система должна удовлетворять санитарным правилам и нормам СанПин 2.2.2./2.4.2198-07;
  4. условия работы средств вычислительной техники (содержание вредных веществ, пыли и подвижность воздуха) должны соответствовать ГОСТ 12.1.005, 12.01.007;
  5. температура окружающего воздуха – 15-35°С;
  6. влажность воздуха – 45-75%.

К оглавлению

Руководство для пользователя:

Навигационная панель.

На навигационной панели есть кнопки для перехода на страницу каталога, справки (сведения о разработчиках и о системе) и вход только для администратора.

К оглавлению

Главная страница (каталог).

Это главная страница с выбором кроссворда для разгадывания.

Можно выбрать один из предложенных кроссвордов, щелкнув по нему левой кнопкой мыши (ЛКМ), будет выполнен переход на страницу разгадывания кроссворда.

Также если на устройстве уже есть сохраненный кроссворд, можно загрузить его, щелкнув ЛКМ по кнопке «Из файла».

Нужно нажать "Select file" и откроется окно для открытия файла с устройства. Нужно выбрать файл и нажать кнопку "Открыть". Загрузить можно только файл с расширением .kros.

К оглавлению

Страница разгадывания.

На данной странице присутствует сетка с кроссвордом, справа указано название кроссворда и находится панель для отображения определения выбранного слова. Снизу расположены кнопки сохранения, скачивания и завершения.

Для того чтобы увидеть определение, нужно щелкнуть ЛКМ по клетке сетки. Будет выделена область на сетке и на панели определения будет показано определение.

Чтобы вписать букву на сетку, нужно щелкнуть ЛКМ по нужной клетке и вписать букву с клавиатуры. Чтобы удалить букву, нужно щелкнуть ЛКМ по нужной клетке и нажать клавишу Backspace на клавиатуре.

Чтобы сохранить кроссворд на данном этапе разгадывания, нужно нажать кнопку «Сохранить». При этом кроссворд не будет сохранен на устройство. Но в будущем можно вернуться на данную страницу и продолжить разгадывания, нажав кнопку "Продолжить разгадывание".

Чтобы скачать кроссворд на устройство, необходимо нажать кнопку «Скачать». Файл будет загружен с расширением .kros.

Чтобы завершить разгадывание и перейти на страницу с результатами, нужно нажать кнопку «Завершить».

К оглавлению

Страница результатов.

На данной странице отображаются ваши результаты разгадывания, и отображается сетка кроссворда с верными ответами.

К оглавлению

Вход.

Если Вы являетесь администратором данного сайта, чтобы перейти на страницу администрирования, нужно нажать кнопку «Вход». Далее нужно ввести логин и пароль и нажать кнопку подтверждения "Войти".
